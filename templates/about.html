About DDoS Detection System

This application uses machine learning to detect and classify DDoS (Distributed Denial of Service) attacks in network traffic.

Dataset Information

The machine learning model was trained on the CICDDoS2019 dataset provided by the Canadian Institute for Cybersecurity (CIC).

This dataset contains network traffic data including both benign (normal) traffic and various types of DDoS attacks:

  • Benign: Normal, non-attack traffic
  • Syn: SYN flood attack
  • UDP: Generic UDP flood attack
  • UDPLag: UDP-based DDoS with lag
  • LDAP: Lightweight Directory Access Protocol-based attack
  • MSSQL: MSSQL-specific DDoS attack
  • NetBIOS: NetBIOS-related DDoS attack
  • Portmap: Portmapper-based DDoS attack

Learn more about the dataset

Machine Learning Model

The system uses a Random Forest classifier to detect and classify DDoS attacks.

Several models were evaluated during development, including:

  • Random Forest
  • K-Nearest Neighbors (KNN)
  • Extra Trees Classifier
  • Multi-Layer Perceptron (MLP) Classifier
  • XGBoost

Random Forest was selected as the best model based on its balanced performance across all metrics:

  • High accuracy in classifying different attack types
  • Good generalization to new data
  • Robust performance across different network conditions
System Architecture

The DDoS Detection System consists of the following components:

Backend
  • REST API: Receives network data and returns predictions
  • Prediction Service: Uses the machine learning model to classify network traffic
  • Database: Stores network data and prediction results
  • WebSocket Server: Provides real-time updates to the UI
Frontend
  • Dashboard: Displays real-time network traffic and attack detection
  • History: Shows historical network data and detections
  • Charts: Visualizes detection statistics and trends
